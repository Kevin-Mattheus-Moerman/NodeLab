## NodeLab 
  NodeLab is a simple MATLAB-package for node-generation and adaptive refinement for meshfree modeling in arbitrary domains. 
### Installation
* Download the package on your PC. 
* Open MATLAB
* Go to the directory 'NodeLab'
* run the script 'setup.m'. This will add different directories of the NodeLab on your MATLAB path â€” only for the current session.
